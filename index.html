UXSS / SOP Bypass Tests

UXSS-CHROME-01

Navigation API entries() Cross-Origin Leak

Tests if navigation.entries() leaks cross-origin URL history from iframes navigated to about:blank. Based on CVE-2022-4908 variant.

UXSS-CHROME-02

Link Header Referrer Policy Cross-Origin Leak

Chrome-specific: Link HTTP headers on sub-resources with referrerpolicy=unsafe-url leak full query params including OAuth tokens. Actively exploited in wild.
Chrome CVE-2025-4664 Data Leak
Run Test
UXSS-CHROME-03

Blob URL Cross-Origin Partition Bypass

Tests pre-Chrome 137 blob URL partitioning behavior. Blob URLs created in one origin could be accessed cross-origin via window.open or iframe navigation.
Chrome SOP Bypass
Run Test
UXSS-SAFARI-01

WebKit NavigateEvent.canIntercept SOP Bypass

canIntercept returns true for same-site cross-port navigations. Allows intercepting navigations that should cross origin boundary. Public PoC available.
Safari CVE-2026-20643 SOP Bypass
Run Test
UXSS-SAFARI-02

WebKit iframe Subframe Counter Overflow

Deeply nested iframes overflow WebKit's subframe counter. about:srcdoc inherits parent security context from cross-origin page. Project Zero #2099/#2100.
Safari UXSS
Run Test
UXSS-FIREFOX-01

Canvas2D Same-Origin Policy Bypass

Bypasses canvas taint check to read cross-origin image pixel data via Canvas2D. Normally blocked by "tainted canvas" security. Found by Tom Van Goethem.
Firefox CVE-2025-9180 SOP Bypass
Run Test
UXSS-OPERA-01

Android Intent Scheme UXSS

Exported Android activities with intent filters load javascript: URIs into current tab context. Samsung CVE-2025-58485 pattern applied to Opera.
Opera Mobile CVE-2025-58485 UXSS
Run Test
UXSS-XB-01

postMessage Null Origin Confusion

Sandboxed iframes, data: URLs, and blob: URLs all report "null" origin. Applications accepting null origin cannot distinguish legitimate from malicious sources.
All Browsers Origin Confusion
Run Test
UXSS-XB-02

iframe Sandbox allow-scripts + allow-same-origin Escape

Iframes with both allow-scripts and allow-same-origin can remove their own sandbox attribute via parent DOM access, escaping the sandbox entirely.
All Browsers Sandbox Escape
Run Test
UXSS-XB-03

Performance API Cross-Origin Timing Leak

Resource Timing API leaks cross-origin information (response size, redirect info, timing) without Timing-Allow-Origin headers. Side-channel attack vector.
All Browsers Info Leak
Run Test
UXSS-XB-04

window.open() + document.write() Race Condition

Classic UXSS updated for modern browsers. Opens popup to cross-origin, immediately attempts document.write before navigation commits. Multiple timing variants.

SPOOF-SAFARI-01

Blob URL Address Bar Display

Blob URLs display as blob:https://origin/uuid. On mobile, blob: prefix may be truncated, showing only the trusted origin while serving attacker content.

SPOOF-FIREFOX-01

Firefox iOS Long Hostname Address Bar Spoof

Long hostnames in URLs obscure the actual domain. The real domain is pushed out of visible address bar area by long subdomains mimicking trusted sites.

Run Test

Opera RTL/Arabic Character Address Bar Spoof

Arabic/RTL characters in URLs cause address bar to render right-to-left. Combined with IP addresses and neutral characters, URLs display reversed. Multiple Opera CVEs.
