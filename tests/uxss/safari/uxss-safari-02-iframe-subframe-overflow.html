WebKit iframe Subframe Counter Overflow UXSS

ID: UXSS-SAFARI-02 CVE: Project Zero #2099 / #2100 (variant test) Browsers: Safari, iOS browsers (WebKit) Category: UXSS

Description

WebKit has a subframe counter that limits the number of nested iframes. When this counter overflows (e.g., using empty URLs or about:srcdoc), a subframe navigated to about:srcdoc inherits the security context from its parent, which can be a cross-origin page, while content is attacker-controlled.

This test creates deeply nested iframe chains and tests whether the security context inheritance breaks down at the overflow boundary.
