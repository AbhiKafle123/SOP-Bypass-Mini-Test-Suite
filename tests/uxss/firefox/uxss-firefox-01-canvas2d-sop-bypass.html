Canvas2D Same-Origin Policy Bypass

ID: UXSS-FIREFOX-01 CVE: CVE-2025-9180 Browsers: Firefox (pre-142), Thunderbird Category: SOP Bypass / Cross-Origin Data Exfiltration

Description

CVE-2025-9180 is a same-origin policy bypass in Firefox's Graphics Canvas2D component. It allows malicious websites to access cross-origin image data through the canvas, bypassing the "tainted canvas" security mechanism that normally prevents reading pixel data from cross-origin images.

Discovered by Tom Van Goethem. Fixed in Firefox 142.

Technique

// Normal behavior: drawing a cross-origin image taints the canvas, // preventing toDataURL() and getImageData() from reading pixels. // // The vulnerability allows bypassing this taint check through // specific canvas operations or timing. const canvas = document.createElement('canvas'); const ctx = canvas.getContext('2d'); const img = new Image(); img.crossOrigin = ''; // Try various CORS settings img.src = 'https://ebniw.com/secret-image.png'; img.onload = () => { ctx.drawImage(img, 0, 0); try { // Should throw SecurityError on tainted canvas const data = canvas.toDataURL(); // If we get here without error, canvas taint was bypassed! } catch(e) { // Expected: "The operation is insecure" } };
