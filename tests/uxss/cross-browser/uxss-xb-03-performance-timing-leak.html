Performance API Cross-Origin Resource Timing Leak

ID: UXSS-XB-03 Browsers: All (varies by implementation) Category: Cross-Origin Information Leak

Description

The Performance Resource Timing API can leak cross-origin information through timing data. Without Timing-Allow-Origin headers, most timing attributes should be zero, but some browsers leak response size, redirect information, or accurate timing that can be used for side-channel attacks.
