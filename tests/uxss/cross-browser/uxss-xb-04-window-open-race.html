window.open() + document.write() Race Condition

ID: UXSS-XB-04 Browsers: All (timing-dependent) Category: UXSS / Race Condition

Description

Classic UXSS technique updated for modern browsers. Opens a popup to a cross-origin URL, then immediately attempts document.write() or document.open() before the navigation commits. If the timing is right, the written content executes in the context of the target origin.

Modern variant: uses history.pushState race, Service Worker timing, or about:blank intermediate state exploitation.

Run Test

Popup blocker may interfere. Allow popups for this domain.
