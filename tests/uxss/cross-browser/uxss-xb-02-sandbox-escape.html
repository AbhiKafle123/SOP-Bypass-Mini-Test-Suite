iframe Sandbox Escape via allow-scripts + allow-same-origin

ID: UXSS-XB-02 Browsers: All Category: Sandbox Escape / SOP Bypass

Description

When an iframe has both allow-scripts and allow-same-origin sandbox flags, the content inside can remove its own sandbox attribute via DOM access to the parent, effectively escaping the sandbox entirely. This is a known design issue in the HTML spec but many developers still misconfigure this.

This test verifies whether the sandbox escape works and demonstrates its impact.
