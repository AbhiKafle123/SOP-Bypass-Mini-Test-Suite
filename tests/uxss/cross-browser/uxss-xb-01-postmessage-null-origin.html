postMessage Null Origin Confusion via Sandboxed Iframes

ID: UXSS-XB-01 Browsers: All (Chrome, Safari, Firefox, Opera) Category: SOP Bypass / Origin Confusion

Description

Sandboxed iframes without allow-same-origin get a null origin. Many applications check event.origin against a whitelist but fail to handle "null" properly. Additionally, data: URLs and blob: URLs in some browsers also report "null" origin, allowing origin confusion attacks.

This tests whether different frame types can impersonate each other's origin via the shared "null" origin value.
