Blob URL Cross-Origin Partition Bypass

ID: UXSS-CHROME-03 Browsers: Chrome (pre-137), Chromium-based Category: SOP Bypass / Cross-Origin Access

Description

Chrome 137 introduced blob URL partitioning by Storage Key. Before this, blob URLs created in one origin context could potentially be accessed cross-origin via window.open() or iframe navigation. This test checks for pre-partition behavior where a blob URL created by one origin can be opened/read by another.

Additionally tests if noopener enforcement on renderer-initiated top-level navigations to cross-site blob URLs is properly applied.
