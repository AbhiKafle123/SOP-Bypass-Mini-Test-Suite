Fullscreen API + Fake Browser Chrome Address Bar Spoofing

ID: SPOOF-OPERA-02 CVE: Opera 2023 (Renwa), CVE-2025-7021 (OpenAI Operator) Browsers: Opera, Opera Mini (Android), Chrome (reduced), Safari (wontfix) Category: URL Spoofing / UI Redressing / BitM

Description

The Fullscreen API hides the browser's real address bar. An attacker can then render a fake browser chrome (address bar, tabs, navigation buttons) showing a trusted URL while the actual content is attacker-controlled. Opera for Android did not show any notification when entering fullscreen, making this particularly effective.

This technique is now used in Browser-in-the-Middle (BitM) attacks. Safari responded to SquareX's disclosure with "wontfix" — claiming the API works as intended.

Run Test

Click the button to enter fullscreen with a fake address bar overlay. Press Escape to exit.
