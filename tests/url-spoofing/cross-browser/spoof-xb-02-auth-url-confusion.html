URL Userinfo (@) Domain Confusion

ID: SPOOF-XB-02 Browsers: Firefox, Opera (most susceptible), all browsers vary Category: URL Spoofing

Description

The URL spec allows user:pass@host format. A URL like https://google.com@evil.com actually connects to evil.com with the username "google.com". Some browsers display the full URL including the userinfo, which can confuse users into thinking they're on google.com.
