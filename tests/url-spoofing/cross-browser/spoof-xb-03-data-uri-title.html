data: URI Tab Title and Content Spoofing

ID: SPOOF-XB-03 Browsers: All (behavior varies) Category: URL Spoofing / Phishing

Description

Data URIs can contain full HTML pages including <title> tags. When opened in a new tab, the tab title shows the attacker's chosen title, and some browsers show data:text/html,... in the address bar which is unfamiliar to most users. Combined with a convincing page design, this enables effective phishing.
