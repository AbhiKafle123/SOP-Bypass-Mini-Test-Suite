Meta Refresh Rapid Redirect Address Bar Race

ID: SPOOF-XB-01 Browsers: All (timing-dependent) Category: URL Spoofing / Race Condition

Description

Rapid <meta http-equiv="refresh"> redirects can cause the address bar to lag behind the actual page content. The page shows content from one URL while the address bar still displays a previous URL in the redirect chain.
