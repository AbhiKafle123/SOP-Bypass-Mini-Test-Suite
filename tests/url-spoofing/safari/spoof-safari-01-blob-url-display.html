Blob URL Address Bar Display Spoofing

ID: SPOOF-SAFARI-01 Browsers: Safari, Chrome, Firefox (varies) Category: URL Spoofing

Description

Blob URLs display as blob:https://origin/uuid in the address bar. If the origin is a trusted domain, the address bar shows what appears to be a legitimate URL while serving completely attacker-controlled content. This is especially effective on mobile where the blob: prefix may be truncated.
