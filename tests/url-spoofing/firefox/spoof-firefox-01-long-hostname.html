Firefox iOS Long Hostname Address Bar Spoofing

ID: SPOOF-FIREFOX-01 CVE: CVE-2025-23109 Browsers: Firefox for iOS (pre-134) Category: URL Spoofing

Description

Firefox for iOS versions below 134 allowed attackers to exploit long hostnames in URLs, obscuring the website's actual host. By using an extremely long subdomain, the real domain is pushed out of the visible portion of the address bar.

Technique

// The attack uses a domain with very long subdomains: // https://secure-login-google-com-verify-account.very-long-sub.attacker.com // // In the address bar, only the beginning is visible: // "secure-login-google-com-verify-acc..." // // The actual domain (attacker.com) is hidden by truncation. // Variant: Use multiple levels of subdomains // https://accounts.google.com.secure-login.verify.attacker.com // DNS Setup required: // *.abhikafle.com.np → points to your server // So: accounts.google.com.secure.abhikafle.com.np works
